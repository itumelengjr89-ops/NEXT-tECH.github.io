NEXTGENTECH — Support Agent Portal

Central resource for Tech Support agents

Purpose

This portal provides Tech Support agents with the forms, procedures and project documentation needed to respond to support requests consistently. Use the "Support Request Log" to record incoming calls, follow the troubleshooting checklist, and consult the escalation procedure when you need to escalate incidents.

Open Support Request Log Troubleshooting Questions Escalation Procedure Project Plan

How to use this Website

  1. When a Global Sales Consultant calls, open Support Request Log, fill in required fields and click Save & Log. The Date & Time will auto-fill but you may edit if needed.
  2. Follow the Troubleshooting Questions checklist and record the steps taken in the form's Troubleshooting area.
  3. If the issue cannot be resolved, open the Escalation Procedure page for the correct escalation path and record actions taken in the form before escalating.
  4. All email communication should use: ithelp@techsupport.com

Contact

All help desk email communications: ithelp@techsupport.com
